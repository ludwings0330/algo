9.1 최적화 문제의 실제 답 계산하기
   - 지금까지는 최적화 문제를 풀 때 최적해의 점수만을 계산했습니다. 하지만, 최적해를 직접 계산해야 하는 문제를 만나면 어떻게 해야 할까요?
     ex) 가장 긴 증가 부분 수열의 길이가 아닌, 가장 긴 증가 부분 수열을 알아내야 할때
         - LIS의 경우 lis()가 LIS의 길이 대신 실제 LIS의 원소들을 배열 을 반환하도록 할 수 있습니다.
            하지만 이 경우, 모든 부분 문제마다 최적해를 저장해야하니, 느리고 메모리를 많이 차지하는 데다 번거롭다는 문제가 있습니다.
            🌸 따라서, 대개 동적 계획법을 사용하는 코드에서는 실제 답을 계산하는 과정을 별도로 구현합니다.
              실제 답을 계산하기 위해서는 각 부분 문제마다 어떤 선택지를 택했을 때 최적해를 얻는지를 기록해 두고, 별도의
              재귀 함수를 이용해 각 조각에서 한 선택을 되짚어 가면서 최적해를 생성해 내면 됩니다.
🌹 최적화 문제 답 계산하기 레시지
    1. 재귀 호출의 각 단계에서 최적해를 만들었던 선택을 별도의 배열에 저장해둡니다.
    2. 별도의 재귀 함수를 이용해 이 선택을 따라가며 각 선택지를 저장하거나 출력합니다.
